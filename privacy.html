wallst-bot 개인정보처리방침

최종 수정일: 2026년 5월 20일

1. 수집하는 정보

wallst-bot은 YouTube Data API를 통해 YouTube 채널에 콘텐츠를 업로드하기 위해 Google OAuth 2.0 인증을 사용합니다. 이 과정에서 다음 정보에 접근합니다:

  • YouTube 채널 관리 권한 (업로드, 수정)
  • Google 계정 이메일 주소 (인증 목적)

2. 정보 사용 목적

수집된 정보는 오직 YouTube 영상 자동 업로드 기능 제공을 위해서만 사용됩니다. 제3자에게 판매하거나 공유하지 않습니다.

3. 정보 저장 및 보안

인증 토큰은 로컬 기기에만 저장되며, 외부 서버로 전송되지 않습니다.

4. 제3자 서비스

본 앱은 Google YouTube Data API v3를 사용합니다. Google의 개인정보처리방침은 https://policies.google.com/privacy에서 확인하실 수 있습니다.
